# hub.docker.com-frakw
Additional files such as docker-compose.yaml or Dockerfile for my docker hub repositories
## usage
```console
cd <repo_name>
```
make `.sh` executable
```console
chmod +x build.sh compose.sh enter.sh
```
build docker image
```console
./build.sh
```
compose image into container and start
```console
./build.sh
```
enter container
```console
./enter.sh
```
